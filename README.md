# 🚢 Titanic Data Analysis Project

## 📌 Overview

This project analyzes the famous Titanic dataset to uncover patterns and factors that influenced passenger survival. It includes data cleaning, exploratory data analysis (EDA), and visualization using Python and Power BI.

## 📂 Project Structure

titanic-analysis/
│
├── data/
│   └── Titanic_Cleaned.csv        # Cleaned dataset
│
├── notebooks/
│   └── Titanic.ipynb             # Jupyter Notebook analysis
│
├── reports/
│   └── Titanic_PowerBI.pbix      # Power BI dashboard
│
├── README.md                     # Project documentation
├── requirements.txt              # Python dependencies
└── .gitignore                    # Ignored files

## 📊 Key Features

* Data cleaning and preprocessing
* Handling missing values
* Exploratory Data Analysis (EDA)
* Visualization of survival trends
* Interactive dashboard using Power BI

## 🛠 Tools & Technologies

* **Python**

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn
* **Jupyter Notebook**
* **Power BI**

## 📈 Insights Explored

* Survival rate by gender
* Survival rate by passenger class
* Age distribution of passengers
* Impact of fare and embarkation port
* Correlation between different features

## 📷 Sample Visualizations

* Bar charts of survival rates
